    Since week 2 was a public holiday (Diwali), our next lesson occured on week 3. In this lesson, we learnt how to create our own website using HTML and CSS. Since I had learnt programming languages such as C++ before. I thought that this would be easy. I was wrong. HTML worked very differently from coding. You had divisions, infile CSS, CSS imported from another file, and you didn't use "" to denote text. Instead, whatever was written between ("< p > and < /p >") or ("< h1 >" and < /h1 >") was automatically treated as text. In fact, anything put inside "" was usually used to define functions such as class = "active" or href = "(website link)" as compared to C++ where text was in bracets while definition of functions were in plain text so it took me some time to wrap my heada around it
