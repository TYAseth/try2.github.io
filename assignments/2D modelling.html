    I was also introduced to basic 2D drawing using Autodesk Fusion360 this session. At first, I was worried that this software would be too hard to use. However, upon some experimentation, I realised that the software was incredibly similar to Autodesk inventor, which I had used before as part of my normal modules.

    To start creating a 2D sketch, just simply click on "New Sketch" and then select the plane that you would like to sketch on. Once the sketch is created, you will see a ribbon:




    What you should see just before selecting a plane to sketch on




    What you should be seeing after you create a sketch




    Once you open up the sketch, the system is pretty straightforward. You select the function you want to use ( if you hover above a function that you're not sure about, a small window will pop up with a description of the function ) and then you left click to execute the function. The cursor automatically snaps to fixed points on a grid so you can easily start from (0,0) to make it easier to draw. When drawing lines, the lines will automatically be straight and snap to certain angles as well, which means you can perfectly draw that horizontal line!

    Even for circles, all you need to do is select the centre point and key in the radius you want. It's very simple and hobbyist friendly.

    During our lesson, we practiced making 2D sketches using the 2D software. We were not supposed to extrude them, but I took the liberty to extrude them. I have made them visible in week 5(3D modelling) if you want to take a look!
