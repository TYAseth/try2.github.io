    In week 5, 3D modelling was covered. In 3D modelling, we were required to create multiple 3D models using Fusion 360. At first, I was scared that 3D modelling would be too complicated for me, however, Fusion 360's interface was pretty similar to autodesk inventor's interface, which I had learn to operate as part of my core diploma modules, theredor, it was relatively easy to pick up 3D modelling using fusion 360.

    We were tasked by our teacher to make various 3D models as part of the 3D modelling course. To make a 3D model , simply make a sketch/sketches, click on extrude/revolve and select the sketches to extrude/revolve. Make sure that your sketch is a closed profile and not an open one.
